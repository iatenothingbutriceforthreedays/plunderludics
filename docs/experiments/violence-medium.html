violence medium

attempt at doing something like 'traditional collage' with a bunch of n64 games. (GoldenEye 007, Castlevania 64, Waialae Country Club, Wheel of Fortune, Resident Evil II, Perfect Dark)



inspiration

  • animonaomi collages

techniques

  • simulate key/controller input (e.g. trigger a jump)
  • modify game state values (e.g. num coins)
  • chroma keying
  • cropping, translating, duplicating, mirroring

tools

  • BizHawk
  • unity

  • babycastles show 2022-12-03
