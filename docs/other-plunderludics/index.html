plunderludics other people made

digital

  • Sonic but every time you pick a ring it changes games (link) by Alistair Aitcheson
  • Journey to Lavender Town (link) by Alistair Aitcheson : pokemon mod where the game slowly gets corrupted over time, ending in lavender town
  • Multibowl (link) by Bennett Foddy and AP Thompson : a wario ware like game that uses many different arcade games as a collection of minigames
  • the NES pinball TAS (link) : finding the TAS movie for another game that maximizes the score for NES pinball)
  • the 4 megamans at the same time thing
  • the megaman where it swaps every time you get hit thing
  • Machinima
  • libsm64-unity (link) : decompiled Mario 64 character controller for using in
  • mario in myst (link) : Mario 64 romhack w recreated Myst island
  • Jazztronauts (link) : probably the most closes to a full blown Videogame thing in this list, a gary's mod map in which you scout random other maps from the steam workshop for multiple kinds of items.
  • Three Skeletons (link) by Lawra Suits Clark

machinima

  • GAME VIDEO/ART: A SURVEY (link) : a show collecting a bunch of machinima art

speedrun/challenge

  • randomizers (link) : modification of videogames in which items/places appear in random order, chaning the challenge.
  • archipelago (link) : a multi-game multi-player randomizer, where items are sent from one game to another

Art

  • Miltos Manetas' Art After Videogames (link) : this guy aparently was the first person to make machinima, but also pretty interested in the images generated by videogames as painting. there's an interesting manifesto in his website: "An artist after Videogames should not create anything himself but only extract the hidden notions of the game."

almost-plunderludic

  • Cadence (link) by Baden Pailthorpe
  • Plundercore (link) : from the itch page: "Games made up of stolen pieces from recognizable works that recontextualize those pieces into new and beautiful things, often playing off of their pre-existing meanings and contexts. Transcending shitpost games into legitimately meaningful art. This is not a comprehensive list, or a strict definition.". this will be used to describe things that use the aesthetic of plunderludics, but the games/videogames are made from scratch.
  • octopad (link) by Patrick Lemieux : its an nes, but the controller is split into 8, each with a single button
  • Platform Games (link) by Patrick Lemieux : many different approaches to looking at the original mario bros, inclusing a response to super mario clouds
  • ROM CHECK FAIL (link) by Farbs
  • Many-Worlds Mario (link) by Andi McClure

hardware

  • NESpectre (link) : nes hardware mod that allows network conntected memory modification

non-digital

  • Name Kit (link) by George Brecht
  • Your Name Spelled with Objects (link) by George Maciunas
  • Dread (link) : tabletop RPG that uses Jenga as a tension mechanic
